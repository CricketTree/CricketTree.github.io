pyc 文件反编译

使用方法

在当前目录（pycdc.exe 和 pyc 文件都要在）打开 cmd 终端，执行以下指令

‘ .\pycdc.exe .\xxx.pyc > xxx.txt ‘

出现以下错误的应对方法：

’ Bad Magic ‘：“ 魔数错误 ”，意思是 pyc 文件的 magic number 不对，一般是文件头不对，解决方法是用 010 editor 打开 pyc 文件将文件头更改或者补充

pyc.encrypted 解密

使用方法

将要解密的 pyc.encrypted 文件放到解密文件目录下执行

pycdecryptor.py
